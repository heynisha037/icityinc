# iCity Inc Website

## Overview

The iCity Inc Website is a modern, responsive corporate website designed to showcase the company's services, business solutions, career opportunities, and contact information. The website provides a professional online presence with an intuitive user experience across desktop, tablet, and mobile devices.

## Features

* Responsive and mobile-friendly design
* Modern and user-friendly interface
* Company profile and about page
* Services and business solutions pages
* Career opportunities and recruitment section
* Contact form and inquiry management
* Optimized performance and fast loading
* Cross-browser compatibility
* SEO-friendly structure

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap
* Responsive Web Design Principles

## Installation

1. Clone the repository:

```bash
git clone https://github.com/heynisha037/icityinc.git
```

2. Navigate to the project directory:

```bash
cd icityinc
```

3. Open `index.html` in your browser or run the project using a local server.
